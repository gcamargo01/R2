R2 Service support core implementations.

Here are tree main components:

{@link uy.com.r2.core.SvcCatalog} The catalog of services is unique (singleton), it hast methods to install, remove and update modules.

{@link uy.com.r2.core.ServiceInfo} The service module structure to hold information and status of each module.

{@link uy.com.r2.core.SimpleDispatcher} The module to dispatch and control execution.

{@link uy.com.r2.core.RunningPipeline} The sequence of service modules to run and current status.

{@link uy.com.r2.core.MBeanConfigurator} JMX dynamic MBean to control the system from jconsole.
